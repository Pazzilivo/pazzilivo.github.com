Markdown 是一种轻量级标记语言，创始人为约翰·格鲁伯（John Gruber）和亚伦·斯沃茨（Aaron Swartz）。它允许人们 “使用易读易写的纯文本格式编写文档，然后转换成有效的XHTML(或者HTML)文档”。这种语言吸收了很多在电子邮件中已有的纯文本标记的特性。 Markdown 同时还是一个由 Gruber 编写的 Perl 脚本：Markdown.pl。它把用 markdown 语法编写的内容转换成有效的、结构良好的 XHTML 或 HTML 内容，并将左尖括号 < 和 & 号替换成它们各自的字符实体引用。它可以用作单独的脚本，Blosxom 和 Movable Type 的插件又或者 BBEdit 的文本过滤器.
